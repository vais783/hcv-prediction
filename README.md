# 🧬 HCV Prediction Web App

A Flask-based web application that predicts Hepatitis C using a machine learning model. Users input data through a web interface, and the system provides real-time predictions.

## 🔧 Tech Stack
Python | Flask | HTML | CSS | Machine Learning

## 📁 Structure
app.py | templates/ | static/ | model.pkl

## 🚀 Features
- User input form  
- ML-based prediction  
- Simple web interface  

## ▶️ Run
pip install -r requirements.txt  
python app.py
